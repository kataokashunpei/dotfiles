## Overview

Some of my dotfiles using for dev.

## Usage

- Download & install
```
git clone https://github.com/taiyoslime/dotfiles && cd dotfiles
chmod +x ./install.sh && ./install.sh
```
- Update
```
chmod +x ./update.sh && ./update.sh
```
